CALIBRATED PASSES · PAGE 2

Instrument and close the first loop

PASS 01 · DAYS 0—30

Instrument the problem

PRINCIPAL WORK
Choose one correction class; map intent, state, frames, data lineage, economic consequence, authority, hold, and rollback.
EXIT EVIDENCE
Signed correction contract; replayable baseline; first scorecard.
AUTHORITY
Observe and recommend.
PASS 02 · DAYS 31—60

Close a narrow loop

PRINCIPAL WORK
Build replay/simulation, shadow inference, human review, proposed correction, post-action measurement, and incident review.
EXIT EVIDENCE
Measured before/after result; rollback; traceable decision record.
AUTHORITY
Shadow with human release.

CALIBRATED PASSES · PAGE 3

Stress and install the compounding system

PASS 03 · DAYS 61—90

Stress the boundary

PRINCIPAL WORK
Vary machine state, tool, material, programming style, operators, environment, sensing, and failure modes.
EXIT EVIDENCE
Known limits; calibrated confidence; hold states; qualification gates.
AUTHORITY
Bounded release where proven.
PASS 04 · DAYS 91—120

Install the compounding system

PRINCIPAL WORK
Translate proof into team design, data architecture, model/physics/rule ownership, simulation roadmap, hiring, and portfolio governance.
EXIT EVIDENCE
Correction-class portfolio; learning-admission rules; operating cadence.
AUTHORITY
Qualified local autonomy.
